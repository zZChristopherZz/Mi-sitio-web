Bienvenidos al Taller HTML

Este es un ejemplo de una página web sencilla usando HTML y CSS. Puedes modificar este contenido y agregar más secciones según tus necesidades. Recuerda siempre mantener la accesibilidad y la usabilidad en mente mientras diseñas.

  • !DOCTYPE html>: Declara el tipo de documento como HTML5
  • Elemento raíz que contiene todo el contenido HTML. Atributo lang="en": Especifica el idioma del contenido como inglés.
  • "head Contiene metadatos y configuraciones del documento (no visibles directamente en la página).
  • meta charset="UTF-8": Define la codificación de caracteres como UTF-8 (compatible con la mayoría de idiomas).
  • name="viewport" content="width=device-width, initial-scale=1.0">: Optimiza la visualización para dispositivos móviles ajustando el ancho de la página.
  • rel="stylesheet" href="styles.css"/>: Vincula una hoja de estilos externa para definir el diseño del sitio.
  • title: Define el título del documento que aparece en la pestaña del navegador.
